Steps

  • Insall react

    npx create-react-app ./

  • Install redux

    npm i react-redux @reduxjs/toolkit

  • store.js

    import { configureStore } from "@reduxjs/toolkit"; import cartReducer from './features/cart/cartSlice' export const store = configureStore({ reducer: { cart: cartReducer } })

  • features/cart/cartSlice.js

    import { createSlice } from "@reduxjs/toolkit"; const initialState = { cartItems: [], amount: 0, total: 0, isLoading: true, } const cartSlice = createSlice({ name: 'cart', initialState }); // console.log(cartSlice) export default cartSlice.reducer

  • Install redux devTools in chrome extension
